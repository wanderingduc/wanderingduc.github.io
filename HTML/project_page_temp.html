Current project

My current project is a top-down visual novel written in Java using the Swing framework. WIth this game I wish to implement the possibility to travel between multiple maps, and interract with NPC's. In the current phase of the program I am working on implementing the basic game mechanics such as player movement, movement between maps, NPC interractions and a game-menu system. When this is done I will fix the encapsulation of variables and methods. The next step is to make a graphics package for the game with object textures and character sprites.
